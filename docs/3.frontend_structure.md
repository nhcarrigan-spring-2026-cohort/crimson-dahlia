## Where to Add New Code

New pages → src/components  
Reusable UI pieces → src/components  
Images & media → src/assets  
Global styles → src/index.css  

## Do Not Edit

node_modules – Automatically generated  
package-lock.json – Managed by npm  

## Common Commands

npm install – Install dependencies  
npm run dev – Start development server  
npm run build – Build for production  

## File Explanations

### JSX Files (alphabetical)

App – Controls overall app routing

CreateAccount – Allows users to create an account with Community Aid

Login – Allows existing users to log in and interact with the app

Main – Acts as the main layout/container for the app

### Folders

node_modules – Contains installed third-party packages (auto-generated, do not edit)

public – Stores static files that are served directly by the app

src – Contains the main application source code

assets – Stores images, sounds, and other static resources used in components

components – Holds reusable UI components used throughout the app

### Other Files

eslint.config.js – Configures ESLint rules for code quality and consistency

index.html – Root HTML file where the React app mounts

package-lock.json – Locks dependency versions to ensure consistent installs

package.json – Lists project dependencies and available npm scripts

vite.config.js – Configuration file for the Vite development and build setup
